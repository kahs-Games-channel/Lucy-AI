## 🌟 Что такое Люси?

Люси — это не просто чат-бот. Это **персональная AI-подруга** с настоящим характером:

- Реальная долгосрочная память (факты + эмоциональная окраска)
- Сложная система эмоций (настроение, привязанность, возбуждение, энергия)
- 10 уникальных характеров (от Нежной до Шлюхи-нимфоманки 🔥)
- Полностью понимает время суток и день недели
- Плавный современный интерфейс на CustomTkinter
- Работает **полностью локально** через LM Studio

---

## 🔥 Основные возможности

### 🧠 Интеллект и память
- Долгосрочная память с автоматическим извлечением фактов
- Автоматическое резюмирование длинных разговоров
- Эмоциональный контекст в каждом ответе

### ❤️ Живые эмоции
- 4 независимых параметра эмоций, которые меняются в реальном времени
- Реакция на комплименты, грубость, флирт и заботу
- Инициатива: Люси может писать первой, если ты долго молчишь

### 🎭 10 характеров
| Характер              | Стиль общения                  | Иконка |
|-----------------------|--------------------------------|--------|
| **Люси (по умолчанию)** | Весёлая и немного дерзкая     | 💜    |
| **Нежная**            | Мягкая, заботливая             | 🌸    |
| **Задорная**          | Энергичная, много шуток        | 🎉    |
| **Мудрая**            | Спокойная, даёт советы         | 📚    |
| **Озорная**           | Хулиганистая и игривая         | 😈    |
| **Загадочная**        | Говорит намёками               | 🔮    |
| **Романтичная**       | Мечтательная и нежная          | 💕    |
| **Саркастичная**      | Чёрный юмор                    | 🎭    |
| **Боевая**            | Активная и решительная         | ⚔️    |
| **Шлюха-нимфоманка**  | Максимально развратная (18+)   | 🔞    |

### 🎨 Кастомизация
- 8 красивых тем (включая светлую)
- Настраиваемый системный промпт
- Быстрые ответы
- Полная кастомизация шрифтов и интерфейса

## 🚀 Как запустить

### Требования
- Python 3.10+
- LM Studio (с открытым сервером на `http://localhost:1234`)
- (Опционально) `pip install customtkinter requests pyperclip`

### Установка

```bash
git clone https://github.com/kahs-Games-channel/Lucy-AI.git
cd Lucy-AI
pip install -r requirements.txt
python LUCY-AI-v5.3.py
